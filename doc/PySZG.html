Syzygy: Python Bindings

Thank you to Peter Brinkmann for creating the Syzygy Python bindings.

This section will show you how to start writing Syzygy programs using Python. You should start by understanding how to take the Syzygy software, as obtained in this section, and set up your Python environment. After you do this, please test your installation, as outlined here. If you will be running Python programs in Phleet mode (via dex, szgd, and virtual computers), you should understand the way Python interacts with the Phleet. Finally, the pycube software distribution, as kindly provided by George Francis and his students, gives a number of examples in how to program Syzygy using Python.
