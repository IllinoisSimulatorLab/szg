Syzygy: Downloads

There are 2 source code packages and one application data package available for download. These provide a complete Syzygy programming environment, both in C++ and Python.

More material is available, including additional source code, libraries, precompiled Syzygy SDKs (which we use with visualization classes here at UIUC), complete snapshots of the sample application data, some motion capture data along with viewers, etc. A DVD containing all this will be mailed upon request.

  1. szg-1.0: The source code for the Syzygy library plus sample applications and documentation. Also contains Python bindings and examples.
  2. szg_app_template: A template project useful for cross-platform C++ projects using Syzygy.
