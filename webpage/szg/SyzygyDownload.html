Syzygy: Software Download

There are 6 source code packages and one application data package available for download. Please do not mix these with earlier versions as they are incompatible.

More material is available, including additional source code libraries, precompiled Syzygy SDKs (which we use with visualization classes here at UIUC), complete snapshots of the sample application data, some motion capture data along with viewers, etc. These are available upon request by e-mailing:

Contact: Benjamin Schaeffer
schaeffr@uiuc.edu

  1. szg-0.7: The source code for the Syzygy library plus sample applications and documentation.
  2. py-szg-0.7: Python bindings for much of the Syzygy API.
  3. pycube: Sample Python applications using Syzygy. Thanks George Francis and students!
  4. vmat-0.7: A realtime physics library used in some of the applications in szgdemo.
  5. szgdemo-0.7: Additional sample applications.
  6. szgdata-0.7: Data for some of the sample applications. Due to space constraints, this cannot be everything. The remaining data is available upon request.
  7. szg_app_template: A template project useful for cross-platform C++ projects using Syzygy.
